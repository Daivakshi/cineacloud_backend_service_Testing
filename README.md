## Description

[Cineacloud](https://cineacloud.com) allows content creators to organize their ideas, write their scripts, plan, budget,
collaborate and distribute their work. All on one platform.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
